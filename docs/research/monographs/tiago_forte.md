# Monografia: Tiago Forte & AI-Augmented CODE

## Core Philosophy: Building a Second Brain (BASB)
Tiago Forte ha reso popolare il sistema BASB per gestire il sovraccarico informativo. Con l'avvento dell'IA, il suo focus si è spostato sulla **Distillazione**: l'IA non deve sostituire il pensiero, ma amplificarlo riducendo il rumore.

## Framework Centrale: CODE
- **Capture (Cattura)**: Salva ciò che "risuona", non tutto.
- **Organize (Organizza)**: Usa il sistema **PARA** (Projects, Areas, Resources, Archives) per agire sulle informazioni.
- **Distill (Distilla)**: Estrai l'essenza (l'IA eccelle qui tramite riassunti e tag automatici).
- **Express (Esprimi)**: Crea output basati sulla conoscenza accumulata.

## Applicazione "Agentic"
Nell'ecosistema Checkvist + MCP:
- **Progressive Summarization**: L'IA analizza un ramo di Checkvist e crea un "TL;DR" automatico per facilitare la revisione.
- **PARA Automation**: L'AI suggerisce in quale "Bucket" (Progetto o Area) spostare un nuovo task in base al contesto.
