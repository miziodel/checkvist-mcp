# Research Sources: Teresa Torres & AI-PKM

This document lists the primary sources and articles used to define the agentic workflows for Checkvist + MCP.

## 1. Teresa Torres (Product Talk)

### [How to Build AI Workflows with Claude Code](https://www.producttalk.org/how-to-build-ai-workflows-with-claude-code/)
- **Utility:** Explains the transition from a "messy brain dump" to a structured output using AI instructions.
- **MCP Application:** Defines how we should prompt the MCP to process raw text into Checkvist list items.

### [How to Use Claude Code: A Guide to Slash Commands, Agents, Skills](https://www.producttalk.org/how-to-use-claude-code-features/)
- **Utility:** Covers functional interaction patterns: slash commands, skills, and plug-ins.
- **MCP Application:** Provides a blueprint for designing our MCP tools and command interface.

### [Our OST in Everyday Life: Bringing Order to Chaos](https://www.producttalk.org/2023/07/opportunity-solution-trees-in-everyday-life/)
- **Utility:** Demonstrates hierarchical organization (Opportunity Solution Trees) applied to personal management.
- **MCP Application:** Matches Checkvist's outliner nature perfectly.

### [My 2026 Roadmap: Exploring Whatâ€™s Just Now Possible](https://www.producttalk.org/my-2026-roadmap/)
- **Utility:** Strategic overview of how AI-powered discovery drives high-level planning.

### [Goals, Processes, & Habits: A Recipe for Success](https://www.producttalk.org/goals-processes-habits/)
- **Utility:** Focuses on the functional structure of daily work organization.
